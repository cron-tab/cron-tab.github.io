React Redux 란

  • React에 Redux를 공식적으로 바인딩한 라이브러리 입니다.

  • 리엑트 컴포넌트가 Redux store에서 데이터를 읽고 데이터를 업데이트하기 위해 store에게 actions를 발송할 수 있습니다.

React Redux 설치

<Provider />

<Provider /> 를 통해 Redux store를 app에서 사용할 수 있습니다.

connect()

connect() 는 컴포넌트와 store를 연결해 줍니다.
