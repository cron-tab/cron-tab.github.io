자바스크립트 스크롤 이벤트

스크롤 이벤트

  • document view나 element가 스크롤될 때 이벤트가 발생.
  • 스크롤 이벤트는 빈번히 발생하기 때문에 DOM 수정과 같은 연산이 많은 작업은 하지 않아야 함.
  • requestAnimationFrame, setTimeout, customEvent를 사용하여 이벤트 호출 조절을 추천.
