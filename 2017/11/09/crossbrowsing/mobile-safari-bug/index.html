아이폰 사파리에서 클릭 이벤트가 되지 않을 경우

아이폰 사파리에서는 a link, input을 제외하고 클릭 이벤트에 대해 event delegation을 지원하지 않는다.

아래 글을 읽어 보면 메모리 관리 문제가 있어 일부러 막은것으로 추정된다.

그래서 클릭 이벤트 동작을 시키기 위해서 <a href=‘#’></a> 감싸 우회하면 동작을 한다.
